Adapting LLMs to Uzbek Language

This project focuses on adapting large language models such as Mistral-7B, Mistral-12B, and LLaMA-8B to the Uzbek language. The goal is to improve natural language processing capabilities for Uzbek, enhancing applications in machine translation, text generation, and language understanding.

  • Implemented fine-tuning techniques for large language models
  • Developed a custom dataset of Uzbek text for training
  • Evaluated model performance using various metrics
  • Published preliminary results demonstrating improved performance on Uzbek language tasks
